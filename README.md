# 大学生时间管理系统 java1084

## 项目概述

大学生时间管理系统是一款帮助大学生合理规划、管理时间的软件。基于前后端分离的技术架构，系统提供管理员与用户两种角色，支持时间分类管理、事件数据管理、目标数据管理等功能。

## 技术栈

- **后端：**
  - Java
  - SpringBoot
  - MyBatis
- **前端：**
  - Vue
- **开发工具：**
  - IDEA
  - JDK1.8
  - Maven
  - Node.js

## 系统角色

系统分为以下两个角色：

### 管理员

管理员登录系统后，可操作以下功能模块：

- 个人中心
- 系统公告管理
- 用户管理
- 时间分类管理
- 事件数据管理
- 目标数据管理
- 用户日记管理

### 用户

用户可通过注册账号，登录进入系统，并可操作以下功能模块：

- 首页
- 个人中心
- 系统公告管理
- 事件数据管理
- 目标数据管理
- 用户日记管理

## 运行环境

- **系统环境：** Windows/Mac
- **数据库：** MySQL

## 部署步骤

1. 环境搭建：安装Java、MySQL、Node.js等运行环境。
2. 数据库部署：执行数据库脚本，创建相关表结构。
3. 后端部署：导入后端代码至IDEA，配置好Maven，运行主程序。
4. 前端部署：导入前端代码至Vue开发环境，编译后部署至Web服务器。

## 核心亮点

- **灵活的时间管理：** 帮助用户根据个人需求自定义时间分类和事件管理。
- **多角色支持：** 分别针对管理员和普通用户提供不同功能模块，满足不同用户的需求。
- **前后端分离：** 基于SpringBoot+Vue的技术架构，支持前后端分离，便于维护和扩展。
